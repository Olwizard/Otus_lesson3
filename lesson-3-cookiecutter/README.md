# Lesson 3 Cookiecutter

Project Description

Setup Instructions
1. Clone the repository:
    ```bash
    git clone git@github.com:NickOsipov/lesson-3-cookiecutter.git
    cd lesson-3-cookiecutter
    ```
2. Create a project:
    ```bash
    uv python pin 3.11.11
    uv init
    uv venv
    uv add --requirements requirements.txt
    ```
